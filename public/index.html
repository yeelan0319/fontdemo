How to run this demo?

  1. Please use Chrome to run this demo.
  2. Right click and select Inspect.
  3. Long press the refresh icon near the address bar (with Developer Tool open).
  4. Select Empty Cache and Hard Reload to remove font file cache.
  5. Either type the URL in the input box and hit LOAD PAGE, or select from the examples below.
  6. Repeat clearing cache before loading any new page, otherwise the browser will directly load font files from cache, and metrics will be wrong.
  7. MLFont decreases needed font file size, thus is applicable to all network conditions. However, even we download much less, the page load time might not have a significant difference with superfast network such as Google A, please try connect to GIN-3g instead.

This demo is for Chinese font. For project details, visit our project homepage: go/mlfont
